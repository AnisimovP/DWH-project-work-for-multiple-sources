Цикл, который получает данные о доставках с помощью API-запросов, пока не будет получен пустой ответ (то есть, пока не будут загружены все данные).
Переменная offset используется для определения количества строк данных, которые уже были загружены, чтобы следующий запрос мог начать загрузку данных с этой позиции. 
При каждой итерации цикла запрос отправляется с параметром offset, указывающим, с какой позиции начинать загрузку данных.
Когда получен пустой ответ, цикл завершается, и выполняется коммит транзакции, закрывается курсор и соединение с базой данных. 

Для каждого набора полученных данных выполняется SQL-запрос на добавление данных в таблицу БД.
Перед выполнением запроса данные преобразуются в список списков.
"execute_values" - для выполнения операции вставки сразу нескольких строк.
